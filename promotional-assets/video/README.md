# Promo Video for PocketPilot

This directory is for an optional promotional video for the Google Play Store listing.

## Requirements

- Maximum length: 2 minutes
- Format: MP4
- High quality (minimum 720p)
- Showcases key features of the app
- Includes voiceover or text overlays to explain features

## Suggested Content

1. Introduction to PocketPilot
2. Demonstration of key features:
   - Budget creation and management
   - Expense tracking
   - Financial insights and charts
   - Privacy and security features
3. Call to action to download the app
