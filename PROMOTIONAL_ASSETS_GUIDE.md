# Promotional Assets Guide for PocketPilot

## Required Assets for Google Play Store

### 1. App Icon
- **File**: assets/logo.png (already exists)
- **Format**: PNG
- **Size**: 512x512 pixels
- **Location**: Already configured in pubspec.yaml

### 2. Feature Graphic
**Description**: A promotional banner displayed in the Play Store
**Dimensions**: 1024x500 pixels
**Format**: PNG or JPEG
**Requirements**:
- No text on the graphic (app name will be overlaid by Play Store)
- High-quality, visually appealing image
- Representative of the app's purpose
- Brand colors: #009688 (teal) and white

### 3. Screenshots
**Phone Screenshots** (minimum 2, recommended 4-8):
- Minimum dimension: 320px
- Maximum dimension: 3840px
- Formats: PNG or JPEG
- Required screen orientations: Portrait (recommended)

**Suggested Screenshots**:
1. Home dashboard with budget overview
2. Expense tracking interface
3. Financial charts and insights
4. Budget creation screen
5. Calendar view of expenses

### 4. Promo Video (Optional)
A short video showcasing the app's features (max 2 minutes)

## Asset Creation Guidelines

### Color Scheme
- Primary: #009688 (Teal)
- Secondary: #FFFFFF (White)
- Accent: #4CAF50 (Green for positive financial indicators)
- Warning: #FF9800 (Orange for budget alerts)

### Typography
- Clean, modern fonts that are easy to read
- Sufficient contrast for accessibility

### Design Principles
- Minimalist and clean interface
- Focus on financial data visualization
- Consistent with Material Design guidelines
- Emphasis on usability and clarity

## Localization
All assets should be created with international audiences in mind:
- Avoid text in images
- Use universally understood financial symbols
- Consider cultural differences in financial practices

## File Organization
Store all promotional assets in a separate directory:
```
promotional-assets/
├── feature-graphic.png
├── screenshots/
│   ├── phone-1.png
│   ├── phone-2.png
│   ├── phone-3.png
│   └── phone-4.png
└── video/
    └── promo-video.mp4 (optional)
```

## Accessibility
Ensure all promotional materials meet accessibility standards:
- Sufficient color contrast
- Clear visual hierarchy
- Avoid flashing or rapidly changing elements
- Consider users with color blindness

## Branding Elements
- Consistent use of the PocketPilot logo
- Unified color scheme across all materials
- Professional and trustworthy appearance
- Emphasis on financial security and privacy
