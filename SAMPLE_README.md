# PocketPilot - Sample Code

This repository contains sample code from the PocketPilot personal finance application.

## Sample Code Structure

The sample demonstrates:
- Core Flutter architecture patterns
- State management approaches
- UI component design
- Data modeling examples

## Key Directories

- `lib/main.dart` - Entry point showcasing app initialization
- `lib/models/` - Sample data models (Budget, Transaction, User)
- `lib/pages/` - Representative UI screens
- `lib/widgets/` - Reusable UI components
- `lib/services/` - Sample service implementations
