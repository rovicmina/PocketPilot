# PocketPilot

[![Google Play](https://img.shields.io/badge/Google_Play-Download-brightgreen)](https://play.google.com/store/apps/details?id=com.pocketpilot.app&pcampaignid=web_share)

PocketPilot is a personal finance application designed to help users track expenses, create smart budgets, and visualize spending patterns.

## About This Repository

This repository contains sample code and screenshots from the PocketPilot application. The full application is available on the Google Play Store.

## Features

- Track daily expenses with categorization
- Create smart budgets based on income/spending patterns
- Visualize spending via interactive charts
- Receive intelligent notifications when nearing budget limits
- Sync data securely using Firebase
- Support calendar integration and profile management
- Ensure privacy with encrypted data storage
